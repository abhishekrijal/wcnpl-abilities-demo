# WCNPL Abilities Demo

A demo WordPress plugin to showcase the WP Abilities API for the speaker session at WordCamp Nepal 2026.

## Overview

This plugin demonstrates the new WordPress 6.9 Abilities API by implementing a simple forms system with four key abilities:

1. **Submit Form** - Public ability to submit form entries
2. **Get Submission Details** - Admin ability to retrieve submission data
3. **Count Forms** - Admin ability to count total forms
4. **Count Submissions** - Admin ability to count submissions (with optional filtering)

## Features

- ✅ Complete WordPress 6.9 Abilities API implementation
- ✅ RESTful API endpoints automatically generated
- ✅ JSON Schema validation for inputs and outputs
- ✅ Proper permission callbacks for security
- ✅ WordPress Coding Standards (WPCS) compliant
- ✅ Internationalization ready
- ✅ Database schema with sample data

## Installation

1. Upload the `wcnpl-abilities-demo` folder to your WordPress plugins directory
2. Activate the plugin through the WordPress admin panel
3. The plugin will automatically create the necessary database tables
